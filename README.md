# ITNARY-BUILDER

🎬 A Modern & Cinematic Itinerary Builder Application

Create beautiful, interactive travel itineraries with smooth animations, stunning visuals, and seamless user experience.

## 🌟 Features

- 🗺️ Interactive map integration
- 🎨 Modern cinematic UI with animations
- 📅 Drag-and-drop itinerary planning
- 🎬 Smooth transitions and visual effects
- 📍 Location-based recommendations
- 💾 Save and share itineraries
- 📱 Fully responsive design

## 📁 Project Structure

```
ITNARY-BUILDER/
├── frontend/          # React + Vite application
├── backend/           # Node.js/Express API
├── docs/              # Documentation
└── README.md
```

## 🚀 Quick Start

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run dev
```

## 🛠️ Tech Stack

**Frontend:**
- React 18
- Vite
- TypeScript
- Tailwind CSS
- Framer Motion (animations)
- Zustand (state management)
- React Query (data fetching)

**Backend:**
- Node.js
- Express.js
- MongoDB
- JWT Authentication
